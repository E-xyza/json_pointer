# JsonJsonPointer

Toolkit for JSONPointer.

## Installation

The package can be installed by adding `json_pointer` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:json_pointer, "~> 0.1.0"}
  ]
end
```

## Documentation

Documentation can be found at https://hexdocs.pm/json_JsonPointer.
